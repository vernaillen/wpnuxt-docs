## Check query in GarphiQl IDE

## 

Create [movies].wql and put it in /queries folder in your project

```
fragment Movie on Movie {
  ...ContentNode
  ...NodeWithFeaturedImage
  content
  title
  editorBlocks {
    name
  }
}
query Movies($limit: Int = 10) {
  movies(first: $limit) {
    nodes {
      ...Movie
    }
  }
}
```


```
const { data } = await useGraphqlQuery('Movies')
```
